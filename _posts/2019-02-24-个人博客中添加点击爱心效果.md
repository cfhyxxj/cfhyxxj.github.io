​    

下载[love.js](https://github.com/cfhyxxj/cfhyxxj.github.io/blob/master/js/love.js)

将```love.js```文件移动到如下目录:

```
/js/love.js
```

然后打开`_layouts/default.html`,找到最后一行,有`</body>`的前面,添加调用

```
<!--鼠标点击爱心动画-->
<script type="text/javascript" src="/js/love.js"></script>
```
