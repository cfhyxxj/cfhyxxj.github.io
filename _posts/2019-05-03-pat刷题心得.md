### `cin输入的时候参数最好不一样，不然会出现bug:`

注释部分为之前的代码，最后忙活了 半天才找出bug！！！

cin可以输入相同的参数，但是后面初始化源结点的dis，num，w都需要用到c1,注释部分的代码改变了c1原始的值。

```
int n,m,c1,c2,c,a,b;
    cin>>n>>m>>c1>>c2;
    for(int i=0;i<n;i++)
        cin>>weight[i];
    for(int i=0;i<m;i++){
        cin>>a>>b>>c;      //cin>>c1>>c2>>c;
        e[a][b]=e[b][a]=c; //e[c1][c2]=e[c2][c1]=c;
    }

    dis[c1]=0;
    num[c1]=1;
    w[c1]=weight[c1];

```
